Legenda:
- [FRAME]: Container / Agrupamento
- [TEXT]: Elemento de texto
- [INSTANCE]: Instância de componente
- [COMPONENT]: Componente reutilizável
- [GROUP]: Grupo de elementos
- [RECTANGLE]: Elemento retangular (div, button, etc)
